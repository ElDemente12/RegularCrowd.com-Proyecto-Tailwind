# Regular Crowd - Official Website

Bienvenido al repositorio oficial de la página web de **Regular Crowd**, una banda de rock alternativo.

## Acerca del Proyecto

Esta es la página web oficial de Regular Crowd, desarrollada con React y Tailwind CSS. El sitio incluye información sobre la banda, discografía, miembros, noticias, tienda de merchandising y un press kit para medios.

## Instalación y Configuración

### Requisitos Previos

- **Node.js**: Descarga e instala Node.js desde [nodejs.org](https://nodejs.org/)

### Pasos de Instalación

1. **Clona el repositorio** (o descarga el código):
   ```bash
   git clone https://github.com/Esteban200603/RegularCrowd.com.git
   cd RegularCrowd.com
   ```

2. **Instala las dependencias**:
   ```bash
   npm install
   ```

3. **Inicia el servidor de desarrollo**:
   ```bash
   npm run dev
   ```

4. **Abre tu navegador** y visita `http://localhost:5173`

¡Listo! La aplicación debería estar corriendo en tu navegador.

## Estructura del Proyecto

```
src/
├── Components/     # Componentes reutilizables (Header, Footer, Form, etc.)
├── Pages/          # Páginas principales (Home, Discography, Members, etc.)
├── assets/         # Imágenes y recursos estáticos
└── index.css       # Estilos globales
```

## Tecnologías Utilizadas

- **React** - Librería de JavaScript para construir interfaces de usuario
- **Vite** - Herramienta de desarrollo rápida
- **Tailwind CSS** - Framework de CSS para diseño
- **Framer Motion** - Librería de animaciones
- **React Router** - Navegación entre páginas
